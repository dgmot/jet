Timeline Area

This view has the purpose of demonstrating
how two components can be combined together.

This application contains a set of views.
Each view contains a set of components.

This is a timeline for the 19th century scientific
events. To use it, simply click the timeline frame or
the timeline overview bar and drag it to the right or left.

When you have dragged the timeline, and new events are available
on the timeline frame, this component will be updated with the current
timeline events, which are added over the accumulated viewed timeline events.
